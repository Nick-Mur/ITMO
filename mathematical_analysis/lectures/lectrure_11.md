## Введение

Мы начали изучение новой темы — **производной и дифференциала** функции. В этой лекции мы ввели понятие производной как предела отношения приращения функции к приращению аргумента. Это можно представить так: стоя в точке x₀, мы смещаемся на Δx вправо или влево и наблюдаем, как изменяется значение функции. Разница между новым и старым значениями функции — это приращение функции Δf, а разница в аргументе — это приращение аргумента Δx. Их отношение и есть средняя скорость изменения функции на этом промежутке. Переходя к пределу при Δx → 0, мы получаем производную функции в точке x₀.

Геометрически это выглядит так: мы проводим секущую через две близкие точки графика функции, а в пределе, когда точки сливаются, секущая превращается в касательную. Производная показывает тангенс угла наклона касательной к графику функции в точке x₀. В случае бесконечной производной касательная становится вертикальной.

Дифференциал функции представляет собой линейную часть приращения функции. Если приращение функции можно разложить на линейную часть по Δx и бесконечно малое более высокого порядка, то функция называется дифференцируемой в точке x₀.

Мы установили, что дифференцируемость функции в точке влечёт её непрерывность в этой точке, но обратное неверно: функция может быть непрерывной, но не дифференцируемой в точке.

Мы также рассмотрели основные правила дифференцирования: производные от суммы, произведения и частного функций, доказали эти правила и научились применять их для вычисления производных сложных функций.

## Определение Производной

**Определение:** Пусть функция f определена на промежутке (a, b), и x₀ ∈ (a, b). Производной функции f в точке x₀ называется предел (если он существует):

f′(x₀) = lim Δx→0 [Δf / Δx] = lim Δx→0 [f(x₀ + Δx) − f(x₀)] / Δx.

Где Δf — приращение функции, Δx — приращение аргумента.

### Геометрический смысл производной

Геометрически производная функции в точке x₀ представляет собой тангенс угла наклона касательной к графику функции в этой точке.

- **Секущая:** Прямая, проходящая через две близкие точки графика функции.
- **Касательная:** Прямая, являющаяся пределом секущей при сближении точек.

**Пример:** Если функция имеет бесконечную производную в точке, то касательная становится вертикальной.

### Примеры вычисления производных

#### Пример 1: Производная f(x) = x² в точке x₀

Вычислим производную по определению:

f′(x₀) = lim Δx→0 [(x₀ + Δx)² − x₀²] / Δx = lim Δx→0 [2x₀Δx + (Δx)²] / Δx = lim Δx→0 [2x₀ + Δx] = 2x₀.

#### Пример 2: Производная f(x) = √x в точке x₀ > 0

f′(x₀) = lim Δx→0 [√(x₀ + Δx) − √x₀] / Δx.

Умножим числитель и знаменатель на сопряжённое выражение:

f′(x₀) = lim Δx→0 [Δx] / [Δx × (√(x₀ + Δx) + √x₀)] = lim Δx→0 1 / [√(x₀ + Δx) + √x₀] = 1 / (2√x₀).

## Дифференциал Функции

**Дифференциалом функции** называется главная линейная часть приращения функции.

**Определение:** Функция f называется **дифференцируемой** в точке x₀, если её приращение Δf можно представить в виде:

Δf = AΔx + o(Δx), при Δx → 0,

где A — постоянное число, а o(Δx) — бесконечно малая величина более высокого порядка по сравнению с Δx.

**Дифференциал функции** определяется как:

df = AΔx.

### Связь дифференциала с производной

**Теорема:** Функция f дифференцируема в точке x₀ тогда и только тогда, когда у неё существует конечная производная в этой точке, и при этом:

A = f′(x₀).

**Доказательство:**

1. **Необходимость:** Если f дифференцируема, то:

   Δf = f′(x₀)Δx + o(Δx).

   Делим на Δx:

   [Δf / Δx] = f′(x₀) + [o(Δx) / Δx].

   Переходя к пределу при Δx → 0, получаем:

   f′(x₀) = f′(x₀) + 0.

2. **Достаточность:** Если существует f′(x₀), то по определению предела:

   [Δf / Δx] = f′(x₀) + α(Δx), где α(Δx) → 0.

   Тогда:

   Δf = f′(x₀)Δx + α(Δx)Δx = f′(x₀)Δx + o(Δx).

### Геометрический смысл дифференциала

Дифференциал функции df = f′(x₀)dx представляет собой приращение ординаты касательной при изменении абсциссы на dx. Это линейная аппроксимация функции в окрестности точки x₀.

## Дифференцируемость и Непрерывность

### Теорема о непрерывности дифференцируемой функции

**Теорема:** Если функция f дифференцируема в точке x₀, то она непрерывна в этой точке.

**Доказательство:**

Из дифференцируемости:

Δf = f′(x₀)Δx + o(Δx).

При Δx → 0, получаем Δf → 0, следовательно:

lim x→x₀ f(x) = f(x₀),

что и означает непрерывность функции в точке x₀.

### Контрпример: Непрерывная, но не дифференцируемая функция

**Пример:** Функция f(x) = |x| непрерывна во всех точках, но не дифференцируема в точке x = 0.

- **Доказательство:** Вычислим левую и правую производные в точке x = 0:

  - Левосторонняя производная:

    f′₋(0) = lim Δx→−0 [|0 + Δx| − |0|] / Δx = lim Δx→−0 (−Δx) / Δx = −1.

  - Правосторонняя производная:

    f′₊(0) = lim Δx→+0 [|0 + Δx| − |0|] / Δx = lim Δx→+0 Δx / Δx = 1.

  Поскольку левые и правые производные не равны, функция не дифференцируема в точке x = 0.

## Односторонние Производные

**Определения:**

- **Правосторонняя производная:**

  f′₊(x₀) = lim Δx→+0 [f(x₀ + Δx) − f(x₀)] / Δx.

- **Левосторонняя производная:**

  f′₋(x₀) = lim Δx→−0 [f(x₀ + Δx) − f(x₀)] / Δx.

### Критерий существования производной через односторонние производные

**Теорема:** Функция f имеет производную в точке x₀ тогда и только тогда, когда существуют обе односторонние производные в этой точке и они равны:

f′(x₀) = f′₊(x₀) = f′₋(x₀).

**Доказательство:**

Следует напрямую из определения производной и свойств односторонних пределов.

## Основные Правила Дифференцирования

Пусть функции f и g дифференцируемы в точке x₀.

### Производная суммы

(f + g)′(x₀) = f′(x₀) + g′(x₀).

**Доказательство:**

Δ(f + g) = Δf + Δg.

Делим на Δx:

[Δ(f + g) / Δx] = [Δf / Δx] + [Δg / Δx].

Переходим к пределу:

(f + g)′(x₀) = f′(x₀) + g′(x₀).

### Производная произведения

(f ⋅ g)′(x₀) = f′(x₀)g(x₀) + f(x₀)g′(x₀).

#### Доказательство правила производной произведения

Рассмотрим приращение произведения:

Δ(f ⋅ g) = f(x₀ + Δx)g(x₀ + Δx) − f(x₀)g(x₀).

Раскроем скобки:

Δ(f ⋅ g) = [f(x₀) + Δf][g(x₀) + Δg] − f(x₀)g(x₀) = f(x₀)Δg + g(x₀)Δf + ΔfΔg.

Делим на Δx:

[Δ(f ⋅ g) / Δx] = f(x₀)[Δg / Δx] + g(x₀)[Δf / Δx] + [ΔfΔg] / Δx.

При Δx → 0, последнее слагаемое стремится к нулю, так как произведение двух бесконечно малых величин порядка o(Δx).

Переходим к пределу:

(f ⋅ g)′(x₀) = f(x₀)g′(x₀) + f′(x₀)g(x₀).

### Производная частного

Если g(x₀) ≠ 0, то:

(f / g)′(x₀) = [f′(x₀)g(x₀) − f(x₀)g′(x₀)] / [g(x₀)]².

#### Доказательство правила производной частного

Рассмотрим приращение частного:

Δ(f / g) = [f(x₀ + Δx) / g(x₀ + Δx)] − [f(x₀) / g(x₀)].

Приведём к общему знаменателю:

Δ(f / g) = {[f(x₀) + Δf]g(x₀) − f(x₀)[g(x₀) + Δg]} / [g(x₀)g(x₀ + Δg)].

Числитель преобразуется в:

[f(x₀) + Δf]g(x₀) − f(x₀)[g(x₀) + Δg] = f(x₀)g(x₀) + Δf g(x₀) − f(x₀)g(x₀) − f(x₀)Δg = Δf g(x₀) − f(x₀)Δg.

Делим на Δx:

[Δ(f / g) / Δx] = [Δf g(x₀) − f(x₀)Δg] / [Δx ⋅ g(x₀)g(x₀ + Δg)].

При Δx → 0, знаменатель стремится к [g(x₀)]², а числитель содержит разности, которые позволяют перейти к производным.

Переходим к пределу:

(f / g)′(x₀) = [f′(x₀)g(x₀) − f(x₀)g′(x₀)] / [g(x₀)]².

## Производная Композиции Функций (Правило Цепочки)

**Теорема:** Пусть функции f(u) и u = g(x) дифференцируемы в соответствующих точках, тогда сложная функция y = f(g(x)) дифференцируема, и её производная равна:

dy/dx = (dy/du) ⋅ (du/dx).

### Доказательство правила цепочки

Рассмотрим приращение сложной функции:

Δy = f(g(x₀ + Δx)) − f(g(x₀)) = f(u₀ + Δu) − f(u₀),

где Δu = g(x₀ + Δx) − g(x₀).

Поскольку f дифференцируема в точке u₀:

Δy = f′(u₀)Δu + o(Δu).

А g дифференцируема в точке x₀:

Δu = g′(x₀)Δx + o(Δx).

Подставляем Δu в выражение для Δy:

Δy = f′(u₀)[g′(x₀)Δx + o(Δx)] + o(Δu).

Замечаем, что o(Δu) = o(Δx), так как Δu → 0 при Δx → 0.

Получаем:

Δy = f′(u₀)g′(x₀)Δx + o(Δx).

Делим на Δx и переходим к пределу:

dy/dx = f′(u₀)g′(x₀).

### Дифференциал сложной функции

Дифференциал сложной функции выражается как:

dy = f′(u)du = f′(g(x))g′(x)dx.

---

## Производная Обратной Функции

**Теорема:** Пусть функция y = f(x) строго монотонна и дифференцируема в точке x₀, причём f′(x₀) ≠ 0. Тогда обратная функция x = f⁻¹(y) дифференцируема в точке y₀ = f(x₀), и её производная равна:

(f⁻¹)′(y₀) = 1 / f′(x₀).

### Доказательство теоремы о производной обратной функции

Рассмотрим функцию y = f(x) и её обратную x = f⁻¹(y).

Приращение функции:

Δy = f(x₀ + Δx) − f(x₀) = f′(x₀)Δx + o(Δx).

Так как функции взаимно обратны:

Δx = f⁻¹(y₀ + Δy) − f⁻¹(y₀).

Тогда:

[Δx / Δy] = 1 / f′(x₀) + o(1).

Переходя к пределу при Δy → 0:

(f⁻¹)′(y₀) = lim Δy→0 [Δx / Δy] = 1 / f′(x₀).

## Таблица Производных

1. **Производная константы:**

   (C)′ = 0.

2. **Производная степенной функции:**

   (xᵅ)′ = α x^(α − 1), x > 0.

3. **Производная экспоненты:**

   (eˣ)′ = eˣ.

4. **Производная логарифма:**

   (ln x)′ = 1 / x, x > 0.

5. **Производная синуса и косинуса:**

   (sin x)′ = cos x, (cos x)′ = −sin x.

6. **Производная тангенса и котангенса:**

   (tan x)′ = 1 / cos² x, (cot x)′ = −1 / sin² x.

7. **Производная арксинуса и арккосинуса:**

   (arcsin x)′ = 1 / √(1 − x²), (arccos x)′ = −1 / √(1 − x²), |x| < 1.

8. **Производная арктангенса и арккотангенса:**

   (arctan x)′ = 1 / (1 + x²), (arccot x)′ = −1 / (1 + x²).

### Вывод производных основных функций

#### Производная синуса

**Цель:** Доказать, что (sin x)′ = cos x.

**Доказательство:**

Рассмотрим приращение:

Δf = sin(x + Δx) − sin x.

Используем формулу разности синусов:

Δf = 2 cos(x + Δx ⁄ 2) sin(Δx ⁄ 2).

Делим на Δx:

[Δf / Δx] = 2 cos(x + Δx ⁄ 2) [sin(Δx ⁄ 2) ⁄ Δx].

При Δx → 0:

[sin(Δx ⁄ 2) ⁄ (Δx ⁄ 2)] → 1,

cos(x + Δx ⁄ 2) → cos x.

Следовательно:

(sin x)′ = cos x.

#### Производная косинуса

**Цель:** Доказать, что (cos x)′ = −sin x.

**Доказательство:**

Δf = cos(x + Δx) − cos x = −2 sin(x + Δx ⁄ 2) sin(Δx ⁄ 2).

Делим на Δx и переходим к пределу аналогично предыдущему примеру.

#### Производная тангенса

(tan x)′ = 1 / cos² x.

**Доказательство:**

tan x = sin x / cos x.

Применяем правило производной частного:

(tan x)′ = [cos x ⋅ cos x + sin x ⋅ sin x] / cos² x = 1 / cos² x.

#### Производная степенной функции

**Цель:** Доказать, что (xᵅ)′ = α x^(α − 1) для x > 0.

**Доказательство:**

Используем определение производной:

f′(x) = lim Δx→0 [(x + Δx)ᵅ − xᵅ] / Δx.

Применяем разложение в ряд или формулу Тейлора для xᵅ:

(x + Δx)ᵅ = xᵅ + α x^(α − 1)Δx + o(Δx).

Следовательно:

f′(x) = α x^(α − 1).

## Параметрически Заданные Функции

Функции могут быть заданы с помощью параметра t:

⎧ x = φ(t),
⎨
⎩ y = ψ(t).

Не всегда такие уравнения задают функцию y(x), так как одному значению x могут соответствовать несколько значений y.

**Пример:** Окружность радиуса R:

⎧ x = R cos t,
⎨
⎩ y = R sin t, t ∈ [0, 2π).

В этом случае зависимость y(x) не является функцией, так как каждому x соответствуют два значения y.

### Производная параметрически заданной функции

**Теорема:** Если функции φ(t) и ψ(t) дифференцируемы, и φ′(t) ≠ 0, то производная y по x выражается как:

dy/dx = ψ′(t) / φ′(t).

### Доказательство теоремы о производной параметрически заданной функции

Рассмотрим функцию y как функцию от x:

- y = ψ(t).
- x = φ(t).

Предполагая, что функция φ(t) строго монотонна и обратима (существует обратная функция t = φ⁻¹(x)), можем записать:

y(x) = ψ(φ⁻¹(x)).

Применяем правило цепочки:

dy/dx = (dψ/dt) ⋅ (dt/dx) = ψ′(t) / φ′(t).
