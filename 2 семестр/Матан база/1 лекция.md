## Определения

1. **Первообразная функции**  
   Функция $F$ называется первообразной для функции $f$ на промежутке $AB$, если  
   $$F'(x)=f(x) \quad \forall\, x\in AB.$$  
   Здесь $AB$ может быть произвольным промежутком (интервал, отрезок, полуинтервал).

2. **Неопределённый интеграл**  
   Неопределённым интегралом функции $f$ на промежутке $AB$ называют множество всех её первообразных, то есть  
   $$\int f(x)\,dx=\{F(x)+C \mid F'(x)=f(x),\; C\in\mathbb{R}\}.$$  
   В более простом виде запись  
   $$\int f(x)\,dx=F(x)+C$$  
   означает, что $F(x)$ — любая первообразная функции $f(x)$, а $C$ — произвольная постоянная.

3. **Линейность неопределённого интеграла**  
   Если $f$ и $g$ имеют первообразные на промежутке $AB$, то для любых констант $α,β\in\mathbb{R}$ выполнено  
   $$\int (α\,f(x)+β\,g(x))\,dx = α\int f(x)\,dx + β\int g(x)\,dx.$$  
   Это свойство следует из линейности операции дифференцирования.

4. **Формула замены переменной (подстановки)**  
   Пусть $u=φ(x)$, где функция $φ(x)$ дифференцируема, а $f(u)$ интегрируема по $u$. Тогда справедлива формула  
   $$\int f(φ(x))\,φ'(x)\,dx = \int f(u)\,du.$$  
   Эта формула позволяет упростить вычисление интеграла путём замены переменной.

5. **Формула интегрирования по частям**  
   Для дифференцируемых функций $u(x)$ и $v(x)$ на промежутке $AB$ справедлива формула  
   $$\int u(x)\,dv(x)=u(x)v(x)-\int v(x)\,du(x).$$  
   Она вытекает из правила дифференцирования произведения функций.

## Формулы

1. **Определение первообразной:**  
   $$F'(x)=f(x) \quad \forall\, x\in AB.$$

2. **Запись неопределённого интеграла:**  
   $$\int f(x)\,dx=F(x)+C.$$

3. **Линейность интеграла:**  
   $$\int (α\,f(x)+β\,g(x))\,dx = α\int f(x)\,dx + β\int g(x)\,dx.$$

4. **Замена переменной:**  
   $$\int f(φ(x))\,φ'(x)\,dx = \int f(u)\,du, \quad \text{где } u=φ(x).$$

5. **Интегрирование по частям:**  
   $$\int u(x)\,dv(x)=u(x)v(x)-\int v(x)\,du(x).$$

## Теоремы

1. **Теорема о единственности первообразных (о константе):**  
   Если $F_1$ и $F_2$ являются первообразными функции $f$ на промежутке $AB$, то  
   $$F_1(x)-F_2(x)=C \quad \forall\, x\in AB,$$  
   где $C$ — некоторая постоянная.

2. **Условие существования первообразной:**  
   Если функция $f$ непрерывна на промежутке $AB$, то существует первообразная для $f$ на $AB$.

3. **Линейность неопределённого интеграла:**  
   Для любых функций $f$ и $g$, имеющих первообразные, и для любых констант $α,β\in\mathbb{R}$ справедливо  
   $$\int (α\,f(x)+β\,g(x))\,dx = α\int f(x)\,dx + β\int g(x)\,dx.$$

## Доказательства теорем

1. **Доказательство теоремы о константе:**  
   Пусть $F_1$ и $F_2$ — первообразные функции $f$ на $AB$. Тогда  
   $$\frac{d}{dx}\Bigl(F_1(x)-F_2(x)\Bigr)=F_1'(x)-F_2'(x)=f(x)-f(x)=0.$$  
   По критерию постоянства (например, по теореме Лагранжа) функция $F_1(x)-F_2(x)$ постоянна, то есть  
   $$F_1(x)-F_2(x)=C \quad \forall\, x\in AB.$$

2. **Доказательство линейности интеграла:**  
   Пусть $F(x)$ и $G(x)$ — первообразные для $f(x)$ и $g(x)$ соответственно, тогда  
   $$\frac{d}{dx}\Bigl(α\,F(x)+β\,G(x)\Bigr)=α\,F'(x)+β\,G'(x)=α\,f(x)+β\,g(x).$$  
   Это означает, что $α\,F(x)+β\,G(x)$ является первообразной для функции $α\,f(x)+β\,g(x)$, что и доказывает линейность.

## Примеры задач

### Пример 1. Нахождение первообразной степенной функции
**Задача:** Найти первообразную для функции $f(x)=x^n$, где $n\ne -1$.  
**Решение:**  
Применяя правило интегрирования степенной функции, получаем  
$$\int x^n\,dx=\frac{x^{n+1}}{n+1}+C.$$

### Пример 2. Вычисление интеграла с заменой переменной
**Задача:** Вычислить интеграл  
$$\int \sin(3x)\cdot 3\,dx.$$  
**Решение:**  
Положим $u=3x$, тогда $du=3\,dx$. По формуле замены переменной имеем  
$$\int \sin(3x)\cdot 3\,dx=\int \sin(u)\,du=-\cos(u)+C=-\cos(3x)+C.$$

### Пример 3. Интегрирование по частям
**Задача:** Вычислить интеграл  
$$\int x\sin x\,dx.$$  
**Решение:**  
Выберем $u=x$ и $dv=\sin x\,dx$. Тогда $du=dx$ и $v=-\cos x$. Применяя формулу интегрирования по частям, получаем  
$$\int x\sin x\,dx=x\cdot(-\cos x)-\int (-\cos x)\,dx=-x\cos x+\int \cos x\,dx=-x\cos x+\sin x+C.$$

### Пример 4. Циклический интеграл
**Задача:** Вычислить интеграл  
$$\int e^x\sin x\,dx.$$  
**Решение (схематично):**  
1. Пусть $u=e^x$, $dv=\sin x\,dx$. Тогда $du=e^x\,dx$, $v=-\cos x$. Получаем  
   $$\int e^x\sin x\,dx=-e^x\cos x+\int e^x\cos x\,dx.$$  
2. Применяем интегрирование по частям к последнему интегралу с $u=e^x$, $dv=\cos x\,dx$ (тогда $du=e^x\,dx$, $v=\sin x$):  
   $$\int e^x\cos x\,dx=e^x\sin x-\int e^x\sin x\,dx.$$  
3. Обозначим исходный интеграл как $I$, тогда  
   $$I=-e^x\cos x + e^x\sin x - I \quad\Rightarrow\quad 2I=e^x(\sin x-\cos x),$$  
   откуда  
   $$I=\frac{e^x(\sin x-\cos x)}{2}+C.$$

### Пример 5. Интегрирование по частям для логарифмической функции
**Задача:** Найти интеграл  
$$\int \ln x\,dx.$$  
**Решение:**  
Выберем $u=\ln x$, $dv=dx$. Тогда $du=\frac{1}{x}\,dx$, $v=x$. По формуле интегрирования по частям получаем  
$$\int \ln x\,dx = x\ln x - \int x\cdot\frac{1}{x}\,dx = x\ln x - \int 1\,dx = x\ln x - x + C.$$

## Остальное

- **Проверка существования первообразной:**  
  Если функция $f$ непрерывна на промежутке $AB$, то по теореме о существовании первообразной на этом промежутке существует функция $F$, удовлетворяющая условию $F'(x)=f(x)$ для всех $x\in AB$.  
  Обратите внимание, что разрывы (особенно разрывы скачком) могут препятствовать существованию первообразной на данном отрезке, поэтому при интегрировании важно анализировать область определения функции.

- **Единообразное обозначение и таблица интегралов:**  
  В конспекте приводится таблица неопределённых интегралов, которая является следствием таблицы производных. Основные интегралы (степенные функции, экспонента, тригонометрические функции, логарифмы) должны быть выучены, поскольку они являются фундаментом для решения более сложных задач.

- **Замена переменной и интегрирование по частям:**  
  Эти методы являются ключевыми для вычисления интегралов сложных функций. Метод замены позволяет свести интеграл к табличному виду, а интегрирование по частям часто применяется в случаях, когда интеграл циклически возвращается к исходному виду, что позволяет решить уравнение относительно интеграла.

- **Циклические интегралы:**  
  При вычислении циклических интегралов (например, $\int e^x\sin x\,dx$) интегрирование по частям применяется несколько раз, что приводит к появлению исходного интеграла в правой части уравнения. Решив полученное уравнение, можно найти значение интеграла.

- **Практические замечания:**  
  - Важно внимательно следить за областью определения функции, особенно при работе с логарифмами и дробями, где может возникать необходимость введения модуля (например, $\ln|x|$).  
  - При применении интегрирования по частям всегда проверяйте корректность выбора функций $u$ и $dv$, чтобы упростить вычисление интеграла.  
  - Постоянная $C$ в неопределённом интеграле отражает тот факт, что все первообразные одной функции отличаются на константу.
